# Python 数据类型全面指南

在 Python 中，数据类型决定了一个变量能做什么，也影响性能、内存占用和代码可读性。掌握常见类型，是学习 Python 的第一步。

## 1. 为什么要理解数据类型

Python 是动态类型语言，变量不需要显式声明类型，但这并不代表它没有类型。比如：

```python
name = "Alice"
age = 18
is_student = True
```

这里的 `name` 是字符串，`age` 是整数，`is_student` 是布尔值。不同类型对应不同操作。

## 2. Python 的内置数据类型

| 类型 | 典型例子 | 特点 |
|---|---|---|
| 数字类型 | `int`, `float`, `bool`, `complex` | 用于数值计算 |
| 字符串 | `"hello"` | 有序、不可变 |
| 列表 | `[1, 2, 3]` | 有序、可变 |
| 元组 | `(1, 2, 3)` | 有序、不可变 |
| 字典 | `{"name": "Alice"}` | 键值对 |
| 集合 | `{1, 2, 3}` | 去重、无序 |
| `None` | `None` | 表示空值 |

## 3. 数字类型

Python 的数字类型主要包括：

```python
age = 18          # int
price = 19.99     # float
is_ok = True      # bool
z = 3 + 4j        # complex
```

其中：

- `int` 用来表示整数。
- `float` 用来表示小数。
- `bool` 本质上是 `True/False`，也可以看作整数 `1/0`。
- `complex` 用于复数运算，日常开发中较少见。

## 4. 字符串

字符串是 Python 中最常见的数据类型之一，适合存储文本信息。

```python
message = "Hello, Python"
print(message[0])      # H
print(message.upper()) # HELLO, PYTHON
```

字符串是不可变的，这意味着你不能直接修改某个字符，而是需要重新生成一个新字符串。

## 5. 列表与元组

### 列表

列表适合存储一组会变化的数据：

```python
scores = [90, 85, 88]
scores.append(95)
print(scores)  # [90, 85, 88, 95]
```

### 元组

元组适合保存不希望被修改的数据：

```python
point = (10, 20)
print(point[0])  # 10
```

列表和元组的核心区别是：

- 列表可变，元组不可变。
- 列表更适合增删改，元组更适合固定配置或常量。

## 6. 字典与集合

### 字典

字典用于表示“键值对”结构：

```python
user = {"name": "Alice", "age": 20}
print(user["name"])  # Alice
```

### 集合

集合用于去重和快速判断成员关系：

```python
nums = {1, 2, 2, 3}
print(nums)  # {1, 2, 3}
```

集合是无序的，不能通过下标访问。

## 7. 可变类型与不可变类型

理解可变和不可变，对写出稳定代码非常重要：

```python
a = [1, 2, 3]
b = a
b.append(4)
print(a)  # [1, 2, 3, 4]
```

这里 `a` 和 `b` 指向同一个列表，所以修改 `b` 也会影响 `a`。

相比之下：

```python
x = "hello"
y = x
x = x + "!"
print(y)  # hello
```

字符串是不可变类型，重新拼接后会生成新的字符串对象。

## 8. 类型转换

Python 支持多种类型转换：

```python
age = "18"
print(int(age))       # 18
print(float(age))     # 18.0
print(str(18))        # "18"
```

常见转换函数包括：

- `int()`：转整数
- `float()`：转浮点数
- `str()`：转字符串
- `list()`：转列表
- `tuple()`：转元组
- `dict()`：转字典

## 9. 实战建议

在实际开发中，建议这样选择数据类型：

- 如果需要存储一组有顺序的数据，优先用列表。
- 如果需要保存固定内容，优先用元组。
- 如果需要按键查找数据，优先用字典。
- 如果需要去重或做集合运算，优先用集合。
- 如果只是表示“没有值”，用 `None`，不要滥用空字符串或 `0`。

## 10. 总结

Python 的数据类型并不复杂，但它们在编程中非常重要。掌握这些基础类型之后，你就能更轻松地阅读别人的代码、写出更清晰的逻辑，也能更好地理解 Python 的底层行为。

如果你刚开始学习 Python，建议把这几类内容反复练习：

1. 数字类型
2. 字符串操作
3. 列表和元组的差异
4. 字典的键值对访问
5. 集合的去重能力

掌握这些基础，你就已经迈出了 Python 学习的第一步。
